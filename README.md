GEE-207 Policlínico Martires de Manatí 
📋 Descripción Esta aplicación ha sido diseñada específicamente para la supervisión y auditoría del GEE-207, un equipo de Tiro Directo. El sistema automatiza el cálculo de la eficiencia energética y el control estricto del combustible en un entorno crítico de salud.
